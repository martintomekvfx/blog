# MDX features

MDX lets you use JSX components inside markdown. This post demonstrates the key features available for writing posts on this blog.

## Syntax-highlighted code blocks

Standard fenced code blocks get automatic syntax highlighting via Shiki:

```typescript
interface Post {
  title: string;
  description: string;
  pubDate: Date;
  tags: string[];
  draft: boolean;
}

function getPublishedPosts(posts: Post[]): Post[] {
  return posts
    .filter((p) => !p.draft)
    .sort((a, b) => b.pubDate.valueOf() - a.pubDate.valueOf());
}
```
